## 二層の決定論

socsim コアは決定論的かつ LLM 非依存であり，LLM は本質的に非決定的である．設計は LLM を 2 つのメカニズムに閉じ込めて擬似決定論化する:

| レイヤ | 構成要素 | 再現性 |
|---|---|---|
| 決定論的 socsim コア | 家計初期化 (パレート賃金・年齢帯)・就業 Bernoulli ドロー・市場調整一様乱数・スケジューリング・財政/金融会計・指標 | seed 固定で bit 単位 (ChaCha20 `SimRng`) |
| 非決定的 LLM レイヤ | 毎月の労働/消費決定・四半期リフレクション | `socsim-llm` `CachingClient` (`hash(prompt+model)` → 応答) + `temperature=0` + 固定 seed |

RNG ストリームは単一 root から派生する: world 初期化に `derive_seed(root, &[0])`，engine (就業ドロー・市場一様乱数) に `derive_seed(root, &[1])`．

## LLM クライアント層 (`llm.rs`)

本番クライアントは

```
CachingClient< Box<dyn LlmClient> >
  backend = FallbackClient< OllamaClient, OpenAiClient >   // Ollama 第一 → OpenAI フォールバック
  cache   = PromptCache (JSON ファイル or in-memory)
```

バックエンドは `Box<dyn LlmClient>` に直接型消去する．`socsim-llm` が `impl LlmClient for Box<T>` を提供する (issue #26) ため専用 newtype は不要である．テストは同じ `EconClient = CachingClient<Box<dyn LlmClient>>` エイリアス越しに `mock::ScriptedClient` を注入する．本クレットは `socsim-llm` を `features = ["live"]` (Ollama + OpenAI バックエンド・`FallbackClient`) で依存する．**`reqwest` 依存は無い**: HTTP は socsim-llm が所有する．

## WorldState とメカニズム

`EconWorld` は `households: BTreeMap<AgentId, Household>` (ソート済みキー → 決定論的 `agent_ids()`) と単一の `MacroEnv` ブロック (物価・利子率・在庫・生産性・税制スケジュール・物価/失業履歴・政策パラメータ) を保持する．1 tick = 1 か月でクロックは `1..=months` を走り，モデルは月インデックスを 0 始まり (`month() = t − 1`) で扱う．

5 メカニズムは 6-phase ループへ次のように対応する:

| Mechanism | Phase | 役割 |
|---|---|---|
| `MacroPolicyMechanism` | Environment | 年次境界の Taylor ルール利子率更新・再分配繰越 |
| `LlmDecisionMechanism` | Decision | 知覚プロンプト → LLM → `{work, consume}` JSON; 就業 `l_i ~ Bernoulli(p^w_i)` を `ctx.rng` で確定 (毎月の唯一の LLM 所在) |
| `MarketClearingMechanism` | Interaction | 生産 `G += Σ l_j·168·A`，需要 `D = Σ p^c_j s_j / P`，不均衡 `φ̄ = (D−G)/max(D,G)`，賃金/物価調整 |
| `FiscalRewardMechanism` | Reward | 累進税 `T(z_i)`・均等再分配・貯蓄更新; 月次マクロ指標の集計・記録 |
| `MemoryUpdateMechanism` | PostStep | 直近 L か月の記憶プール維持・四半期リフレクション (LLM)・就業繰越 |

LLM クライアント・メタデータコレクタ・月次指標バッファは `Rc<RefCell<…>>` でメカニズムと共有する (engine がボックス化メカニズムを所有するため)．run ドライバは実行後にこれらを読み戻し，キャッシュ保存・メタデータ・CSV 出力に使う．

## 指標

`metrics.csv` は 1 月 = 1 行の wide 形式: `month, year, nominal_gdp, price, interest_rate, unemployment_rate, inflation_rate, gini_savings, avg_work_propensity, avg_consume_propensity, redistribution`．Phillips 曲線 (インフレ vs 失業) と Okun の法則 (GDP 成長 vs 失業変化) の Pearson 相関は，このファイルから Python (scipy) で計算する．
