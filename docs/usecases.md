# Use cases

What you can do with this reimplementation of Li et al. (2024) EconAgent.

## 1. Reproduce the emergent macro regularities

Run the model and check that classical macro regularities emerge bottom-up from LLM decisions:

```bash
export OLLAMA_MODEL=llama3.2:latest
cargo run --release -- run --n-agents 100 --months 240 --seed 42
uv run econagent-tools visualize
```

Look at `figures/phillips_okun.png`: a negative Phillips correlation and a strong negative Okun correlation reproduce the paper's qualitative findings. See [Visualization](visualization.md) for the reproduction targets.

For a one-shot batch that runs the headline scenarios, checks the two negative-sign correlation anchors and renders the figures in a single step — including an offline, model-free path — use `reproduce`:

```bash
# offline (no live LLM), short run
cargo run --release -- reproduce --mock --quick --seed 42
uv run econagent-tools reproduce        # macro time series + Phillips/Okun scatter from the latest results
```

`reproduce_summary.json` records the observed Phillips/Okun correlations against the expected negative-sign anchors with PASS/off. The offline mock reproduces the **signs** (the headline); its inflation level is a proxy and does not match the paper's level (a local proxy ≠ the paper's `gpt-3.5-turbo`).

## 2. Study fiscal policy and inequality

Compare fiscal regimes and tax scales and their effect on the savings Gini coefficient:

```bash
cargo run --release -- run --regime none --months 240 --seed 1        # no tax/redistribution
cargo run --release -- run --regime progressive --tax-scale 1.5 ...   # heavier progressive tax
```

Higher tax-scale under the progressive regime should lower the Gini; the `none` regime widens inequality.

## 3. Sensitivity analysis (sweep)

Sweep over agent count × tax-scale × LLM model and aggregate:

```bash
cargo run --release -- sweep --n-agents-values 50,100,300 \
    --tax-scale-min 0.5 --tax-scale-max 1.5 --tax-scale-step 0.5 --runs 3
uv run econagent-tools visualize-sweep
```

The expected finding is that the sign of the macro regularities is robust to N, while tax-scale shifts the Gini. See [CLI](cli.md).

## 4. Offline development and CI (no LLM)

The whole month loop and the output/visualization pipeline run without any live LLM via a scripted mock client:

```bash
cargo run --release --example mock_smoke -- results
uv run econagent-tools visualize
cargo test            # mock-driven; no live LLM
```

## 5. Inspect a run

```bash
uv run econagent-tools show-experiment-settings --results-dir results/latest
```

Prints the configuration and the LLM metadata (model, endpoint, temperature, seed, cache-hit rate) from `run_metadata.json`.
