# CLI (Rust `econagent`)

`cargo build --release` でビルドする．バイナリは `target/release/econagent`．サブコマンドは `run` と `sweep` の 2 つ (`reproduce` は Phase 3 で未実装)．

## LLM 環境変数

LLM プロバイダ順序は «Ollama 第一 → OpenAI フォールバック» で，環境変数で設定する (ハードコードしない):

| 変数 | 既定 | 意味 |
|---|---|---|
| `OLLAMA_HOST` | `http://localhost:11434` | Ollama endpoint (`/api/chat`) |
| `OLLAMA_MODEL` | `llama3.2:latest` | Ollama モデル |
| `OPENAI_API_KEY` | (未設定) | OpenAI フォールバックを有効化 |
| `OPENAI_MODEL` | `gpt-4o-mini` | OpenAI モデル (論文は `gpt-3.5-turbo-0613`) |

Ollama に接続できず OpenAI キーも無い場合，実行は設定エラーで失敗する．ウォームキャッシュがあればモデルは一切呼ばれない．

## `run` — 単一シミュレーション

```bash
cargo run --release -- run \
    --n-agents 100 --months 240 --seed 42 \
    --memory-length 1 --regime progressive --tax-scale 1.0 \
    --llm-temperature 0.0
```

| フラグ | 既定 | 意味 |
|---|---|---|
| `--n-agents` | 100 | 家計数 N |
| `--months` | 240 | 月数 (240 = 20 年) |
| `--memory-length` | 1 | 会話記憶長 L (月) |
| `--regime` | progressive | 財政レジーム: `progressive` / `proportional` / `none` |
| `--tax-scale` | 1.0 | 基準税率への倍率 |
| `--seed` | (ランダム) | 決定論的 socsim コアの root seed |
| `--llm-temperature` | 0.0 | 生成温度 (再現性のため 0) |
| `--llm-seed` | 0 | バックエンド生成 seed |
| `--cache-path` | `.llm_cache/cache.json` | プロンプト→応答キャッシュファイル |
| `--output-dir` | `results` | 出力ベースディレクトリ |

出力は `results/{timestamp}/` に `config.json`・`metrics.csv`・`run_metadata.json`，および `results/latest` シンボリックリンク．

実 LLM パスの軽量スモーク: `--n-agents 4 --months 12 --seed 42`．

## `sweep` — 感度分析

エージェント数 × 税率スケール × LLM モデルを走査し，`sweep_summary.csv` へ独自集計する:

```bash
cargo run --release -- sweep \
    --n-agents-values 50,100,300 \
    --tax-scale-min 0.5 --tax-scale-max 1.5 --tax-scale-step 0.5 \
    --llm-models llama3.2:latest \
    --runs 3 --seed 42
```

| フラグ | 既定 | 意味 |
|---|---|---|
| `--n-agents-values` | `50,100,300` | カンマ区切り N |
| `--tax-scale-min/max/step` | `0.5/1.5/0.5` | 税率スケール格子 |
| `--llm-models` | `llama3.2:latest` | カンマ区切りモデル (セルごとに `OLLAMA_MODEL` も上書き) |
| `--regime` | progressive | sweep 単一の財政レジーム |
| `--months` | 240 | 各実行の月数 |
| `--runs` | 3 | 条件あたりの独立試行数 (派生 seed で独立化) |
| `--seed` | 42 | 基点 seed; 各セルは独立ストリームを派生 |

`sweep_summary.csv` の各行は条件ごとの集計を記録する: 3 年目以降の平均インフレ/失業・最終 GDP・最終 Gini・平均労働/消費傾向・cache-hit 率．出力は `results/{timestamp}_sweep/` に `sweep_config.json` 付き．
